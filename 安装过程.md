安装过程

解压压缩包

```shell
tar -zxvf offline_pkg.tar.gz # 解压压缩包
cd offline_pkg # 进入对应的目录
```



安装系统依赖包

```shell
cd libs # 进入离线包目录
yum -y install ./* # 安装系统依赖包
```



安装python和pip包

```shell
cd ../
cd offline_pkg

# 安装python3.7.5环境
cd python_installer
tar -zxvf Python-3.7.5.tgz
cd Python3.7.5
mkdir /usr/local/python3.7.5
./configure -prefix=/usr/local/python3.7.5
make && make install
ln -s /usr/local/python3.7.5/bin/python3.7 /usr/bin/python3.7
ln -s /usr/local/python3.7.5/bin/pip3 /usr/bin/pip3.7

# 安装python相关第三方依赖包
pip3.7 install --no-index --find-links=./packages/ -r requirements.txt

```
